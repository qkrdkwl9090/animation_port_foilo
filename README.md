<h1 align="left">Nuxt3 Portfilo</h3>

Nuxt3 Portfoilo는 개인적으로 만들어보고 싶었던 페이지나 애니메이션 등을 만들어 배포한 사이트입니다.

Nuxt3, Vue3, Tailwind CSS를 사용하여 개발되었고, 포트폴리오 목적으로 만들어졌습니다.
